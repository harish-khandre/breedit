## Run Locally

Clone the project

```bash
  git clone https://github.com/Harish-Khandre/breedit.git
```

Go to the project directory

```bash
  cd breedit
```

Install dependencies

```bash
  npm install
```

Add the following environment variables to your .env file
```
URI = " "

JWT_SECRET = " "
```
Start the server

```bash
  npm run start
```
